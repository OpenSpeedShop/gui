Open|SpeedShop GUI
an Open|SpeedShop project

A graphical user interface for the Open|SpeedShop project based on Nokia's QT4

Dependencies

This is a preliminary list of dependencies that this project expects to have installed on any client-side system. See the wiki page Installation for more information.

  • Nokia's QT4 [>= 4.6]
  • Python [>= 2.4]

Install

Installation instructions can be found on the wiki page Installation.
